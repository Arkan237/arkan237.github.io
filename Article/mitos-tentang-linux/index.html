Mitos Tentang Linux yang Beredar di Masyarakat

Ditulis pada 5 Juli 2022 oleh Arkan

Banyak orang yang takut menggunakan Linux karena mereka mendengar banyak mitos tentang Linux yang beredar di Masyarakat. Padahal beberapa mitos yang beredar itu tidak sepenuhnya benar. Ada mitos yang sepenuhnya salah, tetapi ada juga yang benar (tergantung kondisi). Apa sajakah mitos itu? Berikut ini beberapa mitos tentang Linux yang beredar dan banyak dipercayai banyak orang

Mitos 1 : Linux itu sulit untuk digunakan

Mitos yang satu ini adalah mitos yang paling terkenal dan banyak dipercaya. Padahal pada kenyataannya tidak seperti itu.

Jujur aja, penulis dulu juga percaya sama mitos kayak gini. Cuma pas penulis mencoba menggunakan Linux, penulis mulai sadar kalau Linux itu tidak sesulit yang dibayangkan. Cara pakainya nggak jauh beda dengan OS sebelah yang paling banyak dipakai.

Sebenarnya mitos ini nggak sepenuhnya salah. Karena memang ada beberapa distro Linux yang lumayan sulit dan tidak "family friendly". Contohnya kayak Arch, Gentoo, LFS. Tapi bukan berarti semua distro Linux tidak "family friendly". Banyak juga distro Linux yang "family friendly". Contohnya kayak Ubuntu, Linux Mint, Elementary OS, Zorin OS, Manjaro, dan masih banyak lagi.

Mitos 2 : Harus hafal perintah dasar Linux

Kalau dipikir-pikir, mtos yang satu ini ada hubungannya dengan mitos yang pertama. Sama juga seperti mitos yang pertama, mitos ini tidak sepenuhnya benar.

Seperti yang penulis katakan sebelumnya, ada beberapa distro yang juga kita harus hafal perintah dasar Linux untuk mengoperasikannya. Namun ada juga distro yang kita tidak perlu menghafal perintah dasar Linux untuk mengoperasikannya. Cara pakainya sama saja seperti menggunakan OS Jendela. Install software bisa lewat Software Center bawaan Ubuntu. Atau bisa juga install lewat file .deb kalau di Ubuntu. Cara install-nya sama saja seperti install software dengan file .exe kalau di OS Jendela.

Meskipun begitu, kalian tetap perlu mempelajari perintah dasar Linux. Memang sekarang ini, kita tidak perlu menyentuh terminal ketika menggunakan Linux untuk aktifitas sehari-hari. Namun tidak ada salahnya mempelajari perintah dasar Linux. Karena siapa tau suatu saat nanti, kita perlu menggunakan terminal. Dan masih ada beberapa hal yang hanya bisa dilakukan melalui terminal di Linux. Kayak contohnya troubleshoting.

Mitos 3 : Aman dari virus

Well, memang Linux itu lebih aman dari virus jika dibandingkan dengan OS Jendela. Tapi bukan berarti Linux aman dari virus. Masih ada kemungkinan Linux akan terserang virus. Namun persentasenya lebih kecil dibandingkan dengan OS Jendela.

Kebanyakan virus yang dibuat untuk Linux biasanya hanya menyerang Linux server. Mengapa demikian? Ya karena banyak perusahaan besar yang menggunakan Linux dalam server mereka. Sehingga para hacker akan mendapatkan keuntungan yang lebih besar jika menyerang Linux server. Mereka jarang menyerang Linux desktop karena kebanyakan penggunanya hanya orang biasa yang tidak mempunyai banyak data yang berharga. Sehingga hanya sedikit keuntungan yang mereka dapatkan jika menyerang Linux desktop.

Meskipun begitu, kalian tetap harus berhati-hati karena masih ada kemungkinan Linux desktop terserang virus.

Mitos 4 : Tidak bisa bermain game di Linux

Kata siapa nggak bisa main game di Linux? Bisa kok main game di Linux. Ada juga beberapa game populer yang support Linux. Kayak contohnya Dota 2, Minecraft, dan lain sebagainya. Cuma ya nggak sebanyak OS Jendela. Namun kalian masih bisa mengakalinya dengan menggunakan software yang bernama Proton. Namun agak sedikit ribet untuk meng-install nya. Dan tidak semua game dapat berjalan mulus di Proton. Jadi meskipun kalian bisa bermain game di Linux, pilihan game nya tidak sebanyak OS Jendela.

Mitos 5 : Linux hanya untuk penggunaan server

Seperti yang penulis katakan sebelumnya, banyak perusahaan besar yang menggunakan Linux dalam server mereka. Mungkin dari sini mitos ini muncul. Mitos ini salah secara mutlak. Kalian juga bisa menggunakan Linux untuk penggunaan sehari-hari.

Di luar sana ada banyak distro yang bisa kalian gunakan untuk sehari-hari. Ada juga distro yang menyediakan 2 versi. Yaitu versi desktop dan server. Distro itu tidak lain adalah Ubuntu. Ubuntu desktop ditujukan untuk pengguna biasa, sehingga dalam Ubuntu desktop sudah pre-installed DE (Desktop Enviromntment) yang memudahkan penggunanya untuk mengoperasikannya. Sedangkan Ubuntu server, ditujukan untuk penggunaan server, sehingga dalam Ubuntu server tidak pre-installed DE apapun. Tampilannya hanya berupa CLI. Sehingga penggunanya perlu menghafal perintah-perintah Linux untuk mengoperasikannya.

Mitos 6 : Harus bisa coding untuk menggunakan Linux

Mitos ini penulis dapatkan dari salah satu grup Linux di Telegram. Awalnya ada seorang yang bertanya di grup itu. Kira-kira seperti ini pertanyaannya.

Bang saran distro Linux yang bisa digunakan tanpa perlu bisa ngoding

Entah darimana dia mendengar mitos ini, tapi yang pasti mitos ini juga salah secara mutlak. Distro apapun itu, kalian tidak harus bisa coding untuk menggunakan Linux. Kecuali kalau kalian mau membuat distro kalian sendiri. Tapi kalau cuma sekadar pakai (apalagi cuma untuk penggunaan sehari), tidak harus bisa coding. Namun kalau kalian bisa coding, kalian bisa memperbaiki masalah yang muncul pada sistem kalian atau memodifikasi distro Linux yang kalian gunakan secara mendalam. Atau kalian juga bisa ikut berpartisipasi pada pengembangan Linux.

Kesimpulan

Menggunakan Linux sebenarnya tidak sesulit yang dibayangkan. Banyak distro Linux di luar sana yang lebih ramah untuk pemula. Oleh karena itu, jangan takut untuk mencoba Linux. Kalau kalian mengalami masalah pada distro Linux yang kalian gunakan, ada banyak komunitas Linux yang siap membantu di luar sana.
